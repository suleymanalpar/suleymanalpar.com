# suleymanalpar.com

Dr. Süleyman Alpar'ın kişisel web sitesi.

Tıp, sağlık hukuku ve hikaye anlatıcılığının kesişiminde bilgi ve ilham sunan platform.

## Sayfalar

- **Ana Sayfa** — Tanıtım, hizmetler ve iletişim
- **Hasta Bilgilendirme** — Hasta hakları, sağlık bilgileri ve SSS
- **Öğrenci Kaynakları** — Ders notları, vaka çalışmaları ve kariyer rehberliği
- **Blog & Makaleler** — Sağlık hukuku, tıbbi etik, bilimsel araştırma ve hikaye

## Teknoloji

- Statik HTML / CSS / JavaScript
- Responsive tasarım
- Türkçe / İngilizce dil desteği
- Netlify üzerinde yayında
